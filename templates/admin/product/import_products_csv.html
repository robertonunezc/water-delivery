📝 Notas importantes

  • Crear o actualizar: Los productos se crean o actualizan por nombre + presentación + unidad de medida.
  • Precio base: El campo base_price es el precio general del producto (obligatorio).
  • Precios por cliente: Agrega client_name y client_price para asignar un precio especial a ese cliente. Si el producto no tiene precio especial, deja ambos vacíos.
  • Varios precios para un producto: Repite el producto en varias filas, una por cada cliente.
  • Categoría: Si la categoría no existe, se crea automáticamente. Si no aplica, déjala vacía.
  • Unidad de medida: Usa valores como lt, ml, pieza, kg, mg, g, m, cm, pulg.
  • Errores: Si una fila tiene error, se omite esa fila pero el resto se procesa.
